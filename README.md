Project Overview :-->
                      LearningApp_Deaf_And_MuteProject is a learning platform designed for deaf and mute users to improve communication and education.
                      The app provides sign language learning modules and a Sign Language ↔ English/Gujarati converter for better interaction.
                      It supports inclusivity in education, aligning with the Government of Gujarat’s “Opportunity for All” initiative.
                      The system helps users learn, practice, and translate signs using interactive content, quizzes, and progress tracking.
